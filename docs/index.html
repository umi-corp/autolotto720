✨ 주요 기능

🎫

자동 배정

번호 고민 없이 매주 자동 배정으로 구매. 기본 1매, 최대 5매까지.

⏰

자동 구매

매주 목요일 추첨 전, 설정한 요일·시간에 자동 구매. 앱을 켜지 않아도 동작.

🔔

알림

구매 완료, 당첨 결과를 푸시 알림으로 받아보세요. 서버 없이 로컬에서 동작.

📋

구매 내역

동행복권에서 최근 구매 내역을 자동 조회. 조·번호와 당첨 여부를 한눈에.

🔒

보안

계정 정보는 Android Keystore로 암호화 저장. 서버 전송 없음.

💸

완전 무료

광고 없음. 구독 없음. 서버 비용 없음. 오픈소스.

📱 사용 방법

1

APK 설치

위 버튼으로 APK를 다운로드하고 설치하세요.

2

동행복권 로그인

설정 탭에서 동행복권 계정으로 로그인합니다.

3

구매 설정

구매 탭에서 매수(1~5매)와 구매 요일·시간을 설정합니다.

4

자동 구매 ON

자동 구매를 켜면 끝! 매주 알아서 구매하고 당첨까지 확인합니다.

⚠️ 이 앱은 동행복권(dhlottery.co.kr) 계정이 필요합니다.
예치금이 충분해야 자동 구매가 진행됩니다.
